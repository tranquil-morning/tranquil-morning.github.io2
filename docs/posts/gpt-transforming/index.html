DeepLearning.AI에서 제공하는 ChatGPT Prompt 강좌를 요약한 글입니다. 실제 OpenAI 개발자가 진행하는 강좌이기에, 단순히 이용자의 경험과 감으로 제공되는 prompt 보다 나을 것으로 예상하여 정리합니다.

모든 예시는 GPT 4.0 API로 작성되었습니다.

목차 :

  • 프롬프팅의 원칙
    • 명확하고 구체적인 지침 작성
    • 모델에게 생각할 시간 주기
  • 프롬프트 개발하기
  • GPT 기능 활용
    • Summarizing
    • Inferring
    • Transfroming
    • Expanding

이번 문서에서는 GPT를 활용하여 변환을 하는 방법들에 대해 알아보겠습니다. 번역, 톤 바꾸기, 문법체크를 예시로 알아보겠습니다.
